# AI Travel Planner

A full-stack travel planning application built with modern technologies.

## Features

- **Personalized Itineraries**: Generate custom travel plans based on your preferences.
- **Interactive Maps**: Explore destinations with an integrated map view.
- **Dynamic Content**: Rich destination details and experiences.
- **Modern UI**: Clean, responsive design using Tailwind CSS and shadcn/ui.

## Tech Stack

- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui, Framer Motion
- **Maps**: Leaflet, OpenStreetMap
- **State Management**: TanStack Query
- **Backend (Optional)**: Can be integrated with any REST API.

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd plan-it-today
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Development

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Runs ESLint for code quality checks.
- `npm run test`: Runs unit tests.
